# WeatherApp Project
* Vue 3
* Vuetify
* Vuex
* Auth
* Router Guard


## Install the dependencies
 ```bash
yarn
# or
npm install
```

### Start the app in development mode

* server 
```bash
json-server db.json
```

* frontend 
```bash
npm run dev
```
